# Allow Image Deletion

## Tasks
- Admins should be able to delete images from the image list page or the image edit page
- Only images that are not used in other entities can be deleted
- Check that images are not used in homepage_projects or any homepage versions
- If image is used, notify the user where the image is used and give links to where that entity can be deleted or modified
- Confirm any deletion with a modal
- Delete the image folder and all image files from storage
- Delete the record in the database
